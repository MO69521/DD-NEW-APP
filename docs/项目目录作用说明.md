# 点点穿书新 APP：项目目录作用说明

> 本文按当前仓库实际结构整理，重点标注“应该维护/可改”的代码文件用途，方便后续开发快速定位。  
> 架构基准：Feature First（`core` / `shared` / `features` / `routes`）。

## 1. 根目录（工程入口与配置）

- `README.md`：项目基础说明（初始化模板内容，可继续补充）。
- `analysis_options.yaml`：Dart/Flutter 静态检查规则配置。
- `pubspec.yaml`：依赖、资源（assets）、Flutter 构建配置。
- `pubspec.lock`：依赖锁文件（记录具体版本）。
- `devtools_options.yaml`：DevTools 配置。
- `.gitignore`：Git 忽略规则。
- `.metadata`：Flutter 工程元数据（自动维护）。
- `diandian_chuanshu.iml`：IDE 工程文件（自动维护）。

## 2. 资源目录 `assets/`

### `assets/icons/`
- `search.svg`：搜索栏左侧搜索图标。
- `category.svg`：顶部“分类”按钮图标。
- `arrow_right.svg`：区块右侧“更多/完整榜单”箭头图标。

### `assets/covers/`
- 书籍封面图目录（当前由 `bookstore_preview_data.dart` 中资源路径引用）。

## 3. 核心能力 `lib/core/`

### `lib/core/index.dart`
- `core` 层统一导出入口。

### `lib/core/constants/`
- `app_constants.dart`：全局常量（跨模块共享）。

### `lib/core/services/`
- `service_locator.dart`：服务定位/依赖注入入口（当前可扩展）。

### `lib/core/theme/`（设计系统 Token）
- `app_colors.dart`：颜色 token。
- `app_spacing.dart`：间距 token。
- `app_sizes.dart`：尺寸 token（组件尺寸、占位、指示器等）。
- `app_radius.dart`：圆角 token。
- `app_text_styles.dart`：文字样式 token。
- `app_durations.dart`：动效时长 token。
- `app_theme.dart`：全局主题对象（ThemeData）定义。

## 4. 共享层 `lib/shared/`

### `lib/shared/index.dart`
- `shared` 层导出入口。

### `lib/shared/widgets/`（L1 原子组件）
- `app_text.dart`：统一文本组件封装。
- `app_button.dart`：统一按钮组件封装。
- `app_icon.dart`：统一图标组件（SVG 渲染）。
- `book_cover.dart`：书封图片组件（圆角、边框、尺寸约束）。

### `lib/shared/components/`（L2 组合组件）
- `glass_chip_button.dart`：玻璃态胶囊容器/按钮（搜索框、分类按钮复用）。
- `section_header.dart`：区块标题 + 右侧动作入口。
- `book_card_variants.dart`：书籍卡片变体（竖卡、横卡）与公共文本区。
- `book_grid_card.dart`：网格书卡（封装 `BookCardVertical`）。
- `book_list_tile.dart`：榜单横向书项（封装 `BookCardHorizontal`）。
- `empty_state.dart`：空状态展示组件。

### `lib/shared/layouts/`
- `app_scaffold.dart`：页面基础布局壳（可供多页面复用）。

## 5. 业务模块 `lib/features/`

## 5.1 书城模块 `lib/features/bookstore/`

### `index.dart`
- 书城模块导出入口。

### `domain/entities/`
- `book.dart`：书籍实体、榜单 tab 枚举等领域模型。

### `application/`（状态管理）
- `bookstore_interaction_state.dart`：交互状态（榜单 tab、底部导航选中等）。
- `bookstore_state.dart`：页面聚合状态（猜你喜欢列表、分页加载状态等）。
- `bookstore_cubit.dart`：书城状态与交互逻辑（切 tab、切底部导航、加载更多）。

### `presentation/pages/`
- `bookstore_page.dart`：书城主页面（Sliver 结构、粘性头部、滚动监听、底部导航挂载）。
- `bookstore_preview_data.dart`：书城预览 mock 数据（榜单、推荐、导航数据）。

### `presentation/components/`
- `bookstore_search_bar.dart`：搜索栏组件。
- `bookstore_search_header.dart`：顶部搜索区（搜索栏 + 分类按钮）。
- `bookstore_bottom_nav.dart`：底部导航（视觉层，状态由 application 提供）。
- `bookstore_nav_icon.dart`：底部导航图标渲染。
- `ranking_tabs.dart`：榜单 tab 切换条。
- `ranking_section_header.dart`：榜单区块头部（tab + 完整榜单入口）。
- `ranking_book_grid.dart`：榜单 2 列紧凑书单。
- `ranking_section.dart`：榜单区块聚合组件。
- `book_grid_section.dart`：通用网格区块（标题 + 网格）。
- `editor_pick_section.dart`：编辑推荐区块。
- `guess_like_section.dart`：猜你喜欢区块（用于无限加载场景）。

### `android/`

## 11. 自动生成/临时目录（建议不手改）

- `.dart_tool/`：Flutter/Dart 工具缓存与构建中间态。
- `build/`：编译产物目录。
- `ios/Flutter/ephemeral/`：iOS 构建临时文件（每次构建可能重写）。
- `android/.gradle/`：Gradle 缓存。

---

## 维护建议（简版）

- 新业务优先放在 `features/<name>/data|domain|application|presentation`。
- UI 样式统一走 `core/theme/*` token，避免组件内硬编码。
- 可复用 UI 放 `shared/widgets`（L1）或 `shared/components`（L2）。
- 路由统一在 `routes` 注册，页面内避免直接 `Navigator.push()`。
