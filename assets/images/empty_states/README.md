# empty_states · 空状态插图（三主题公用）

列表 / 页面无数据时的插画，**不随 `THEME` 分包**，三套主题共用一套。

| 文件 | 场景 |
|------|------|
| `empty_bookshelf.png` | 书架为空 |
| `empty_card_pack.png` | 卡包为空 |
| `empty_messages.png` | 消息为空 |

代码路径：`AppSharedAssets.empty*`（[`app_shared_assets.dart`](../../lib/core/theme/app_shared_assets.dart)）。
