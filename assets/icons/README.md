# icons · 图标资源目录（中英对照）

> **文件夹英文名不要改**——`pubspec` 与代码路径依赖它们。本文件仅供查阅。

| 目录 | 中文 | 用途 |
|------|------|------|
| `account_settings` | 账号设置 | 绑定微信 / QQ / 手机 / 抖音等 |
| `book_detail` | 书籍详情 | 加书架、送心、促销标、刷新、讨论点赞等（含按主题分包） |
| `membership` | 会员 | VIP 权益图标 |
| `nav` | 底部导航 | 书城 / 福利 / 伙伴 / 书架 / 我的（按主题分包） |
| `payment` | 支付 | 支付宝 / 微信支付 |
| `profile` | 我的 | 设置、装扮、卡册、阅读历史等入口图标 |
| `ranking` | 排行榜 | 名次角标、返回、分享等 |
| `search` | 搜索 | 搜索入口 / 空态图标 |
| `shared` | 通用 | 跨页面共用：右箭头、下箭头、热门火焰 |
| `welfare` | 福利 | 签到、任务、能量等 |

## 主题分包（仅部分目录有）

`nav/`、`book_detail/` 下另有：

| 子目录 | 中文 |
|--------|------|
| `yellow_dark` | 黄色深色主题 |
| `pink_light` | 粉色浅色主题 |
| `yellow_light` | 黄色浅色主题 |
| `shared`（仅 book_detail） | 该业务内跨主题共用 |

路径约定与加载方式见 [`docs/09_Assets.md`](../../docs/09_Assets.md)。
